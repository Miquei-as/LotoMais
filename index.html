Como usar:

  1. Clique em "Iniciar Câmera" para ativar sua câmera.
  2. Posicione a cartela da Lotofácil dentro da área demarcada.
  3. Clique em "Escanear Cartela" para processar a imagem.
  4. Os números detectados serão marcados automaticamente em VERDE.
  5. Digite os números sorteados no campo indicado.
  6. Clique em "Verificar Números" para destacar em VERMELHO os números que você marcou mas não foram sorteados.
  7. Use "Exportar para Excel" para salvar os resultados.
  8. Use as abas para visualizar os diferentes jogos identificados.
