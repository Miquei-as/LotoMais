Como usar:

  1. Selecione o método de OCR desejado (para APIs reais, insira sua chave)
  2. Clique em "Iniciar Câmera" para ativar sua câmera.
  3. Posicione a cartela da Lotofácil dentro da área demarcada.
  4. Clique em "Escanear Agora" para processar a imagem.
  5. Use as abas para visualizar os diferentes jogos identificados.
  6. Clique em "Copiar Números" para copiar os números selecionados.

Sobre as APIs:

  • Google Cloud Vision: API paga com alto precisão (requer chave API)
  • Azure Computer Vision: API Microsoft com boa precisão (requer chave API)
  • OpenAI GPT-4 Vision: Reconhecimento avançado por IA (requer chave API)
  • OCR Simulado: Funcionalidade básica sem necessidade de API
