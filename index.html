Como usar:

  1. Clique em "Carregar Jogos" para carregar os jogos da imagem
  2. Os números detectados serão marcados automaticamente em VERDE
  3. Digite os números sorteados no campo indicado
  4. Clique em "Verificar Números" para destacar em VERMELHO os números que você marcou mas não foram sorteados
  5. Use "Exportar para Excel" para salvar os resultados
  6. Use as abas para visualizar os diferentes jogos identificados
