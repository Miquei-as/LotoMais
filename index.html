Cartela de Bingo Corrigida

Agora com verificação correta e exportação para Excel

Como usar:

  1. Clique nos números que você marcou em cada jogo (eles ficarão verdes)
  2. Insira os números sorteados separados por vírgula no campo abaixo
  3. Clique em "Verificar" para identificar números marcados que não foram sorteados (eles ficarão vermelhos)
  4. Use "Exportar para Excel" para baixar um arquivo com sua cartela completa

O arquivo Excel conterá todos os 3 jogos com as marcações atuais.
