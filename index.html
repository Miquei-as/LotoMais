Como usar:

  1. Clique em "Iniciar Câmera" para ativar sua câmera.
  2. Posicione a cartela da Lotofácil dentro da área demarcada.
  3. O sistema fará captura automática quando números forem detectados.
  4. Use as abas para visualizar os diferentes jogos identificados.
  5. Clique em "Copiar Números" para copiar os números selecionados.
  6. Use "Parar Câmera" para desativar a câmera quando não estiver usando.
